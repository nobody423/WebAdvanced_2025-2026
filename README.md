Pokédex — Web Advanced Project
Een interactieve single-page applicatie gebouwd met vanilla JavaScript en Vite, die gebruik maakt van de PokéAPI. Gebruikers kunnen Pokémon verkennen, filteren, sorteren en opslaan als favoriet.

Projectbeschrijving
Deze applicatie toont een overzicht van Pokémon opgehaald via de PokéAPI. De gebruiker kan zoeken op naam, filteren op type, sorteren op ID/naam/gewicht en Pokémon opslaan als favoriet. Favorieten worden bewaard via LocalStorage zodat ze beschikbaar blijven na het herladen van de pagina. Nieuwe Pokémon worden automatisch geladen via infinite scroll.

Functionaliteiten

Lijst van 20+ Pokémon met sprite, ID, naam, types, hoogte en gewicht
Zoeken op naam (live terwijl je typt)
Filteren op type (fire, water, grass, electric)
Sorteren op ID, naam of gewicht
Favorieten opslaan en verwijderen
Favorieten bewaard tussen sessies via LocalStorage
Infinite scroll: automatisch meer Pokémon laden bij het scrollen
Formuliervalidatie op het zoekveld
Responsive design

Gebruikte API
PokéAPI — https://pokeapi.co/

Installatie

npm install
npm run dev
npm run build
